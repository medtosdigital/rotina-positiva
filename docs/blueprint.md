# **App Name**: Magic Routine

## Core Features:

- Interactive Routine Display: Visually present the daily schedule as a game, utilizing interactive elements for children to track their progress.
- Gamified Reward System: Implement a digital reward system using virtual coins and stars, motivating children to complete tasks and promoting positive behavior.
- AI-Powered Emotional Analysis: Analyze images uploaded to the 'before' section. Use the results to select an appropriate affirmation from a database of pre-written affirmations. Suggest that affirmation in the 'after' section. This feature acts as a tool.
- Parent Dashboard: Provide a dashboard for parents to customize routines, manage rewards, and track their child's progress.
- Customizable Task Cards: Allow parents to create and customize task cards with their own images and descriptions to fit their child's specific needs and routines.

## Style Guidelines:

- Primary color: Turquoise blue (#40E0D0) for a calming yet vibrant feel.
- Background color: Very light turquoise (#E0FFFF) to complement the primary color.
- Accent color: Golden yellow (#FFD700) for highlighting key elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif to ensure legibility, paired with warmth, especially on longer sections of text
- Use simple, playful icons and emojis to enhance the child-friendly aesthetic.
- Employ large, impactful blocks with rounded edges and soft shadows to create a sense of depth and physicality.
- Incorporate subtle animations and hover effects to enhance user engagement.
